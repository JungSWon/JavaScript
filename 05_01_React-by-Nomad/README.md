##  환경설정 

- `node -v`  설치 확인
- `npm -v` 설치 확인 ( 보통 node와 같이 자동 설치 됨 )
- `git --version` 설치 확인 
- `npm install npx -g`  npx 설치
- IDE 
  - VSC (무료) 
  - intelliJ (유료)
  - 외 JS , React 를 지원하는 에디터

##  사전지식 

- CSS/HTML
  - div, span, flexbox, display block, background color 가 뭔지 안다면 OK
- 약간의 JS 
  - function, variable, class 가 뭔지 안다면 OK
