## 개념
- 다양한 웹 브라우져에서 자바스크립트(Javascript)가 공통되게 잘 작동하기 위해 필요한 표준 규격을 
- ECMA 국제 기구에서 스크립트 표준인 “ECMAScript Standard” 만듦
- Javascript : {코어:[ECMAScript], 모델:[BOM, DOM], ..}
  - BOM : Browser Object Model
  - DOM : Document Object Model
